## 📸 **Adicionando Novas Fotos**

### ✅ **Método Simples:**
1. **Acesse** a pasta `slideshow` no GitHub
2. **Clique** em "Add file" → "Upload files"
3. **Arraste** suas novas fotos
4. **Nomeie** seguindo o padrão (`roupa11.jpg`, `roupa12.jpg`, etc.)
5. **Commit** as mudanças
6. **Aguarde 2-3 minutos** para atualizar

### 🔄 **Atualizando o Código (Se Necessário)**
Se você adicionar mais de 10 fotos, edite o arquivo `index.html`:

1. **Encontre esta seção** no código:
```javascript
const slideshowImages = [
    'slideshow/roupa1.jpg',
    'slideshow/roupa2.jpg',
    'slideshow/roupa3.jpg',
    'slideshow/roupa4.jpg',
    'slideshow/roupa5.jpg',
    'slideshow/roupa6.jpg',
    'slideshow/roupa7.jpg',
    'slideshow/roupa8.jpg',
    'slideshow/roupa9.jpg',
    'slideshow/roupa10.jpg'
];
```

2. **Adicione mais linhas** conforme necessário:
```javascript
const slideshowImages = [
    'slideshow/roupa1.jpg',
    'slideshow/roupa2.jpg',
    // ... fotos existentes ...
    'slideshow/roupa11.jpg',
    'slideshow/roupa12.jpg',
    'slideshow/roupa13.jpg'
    // Adicione quantas quiser
];
```
