## 🚀 Como Atualizar no GitHub

### Passo 1: Substituir o arquivo atual

1. Acesse seu repositório: https://github.com/evozago/lui-bambini-avaliacao
2. **Exclua TODOS os arquivos atuais**:
   - Clique na caixa de seleção no topo para selecionar todos
   - Clique no botão "Delete" (lixeira)
   - Confirme a exclusão
3. **Faça upload do novo arquivo**:
   - Clique em "Add file" > "Upload files"
   - Arraste APENAS o arquivo `index.html` para a área de upload
   - Adicione uma mensagem como "Sistema funcional completo"
   - Clique em "Commit changes"

### Passo 2: Aguardar atualização

- Aguarde 2-3 minutos para o GitHub Pages atualizar
- Acesse: https://evozago.github.io/lui-bambini-avaliacao/
- Teste todas as funcionalidades

## 📱 Como Usar o Sistema

### Sistema de Avaliação

1. **Etapa 1**: Cliente seleciona a vendedora que o atendeu
2. **Etapa 2**: Cliente seleciona como conheceu a loja
3. **Etapa 3**: Cliente avalia de 1 a 5 estrelas e pode deixar comentário
4. **Envio**: Dados são enviados automaticamente para o Google Sheets

### Dashboard de Relatórios

1. Clique no botão "📊 Relatórios" no canto superior direito
2. Visualize estatísticas gerais
3. Veja distribuição por origem dos clientes
4. Analise avaliações por vendedora
5. Clique em "← Voltar" para retornar ao sistema de avaliação

O sistema coleta automaticamente:

- Avaliação de 1 a 5 estrelas

## 🎯 Configuração para iPad (Totem)

1. Abra o Safari no iPad
2. Acesse: https://evozago.github.io/lui-bambini-avaliacao/
3. Toque no botão "Compartilhar" (quadrado com seta)
4. Selecione "Adicionar à Tela de Início"
5. Configure o iPad para não entrar em modo de espera
6. Posicione o iPad em local acessível aos clientes
